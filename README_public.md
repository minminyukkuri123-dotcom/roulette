# ルーレットアプリ 使い方ガイド

このアプリは、イベントや抽選会で使えるシンプルで高機能なルーレットです。

## 基本的な使い方

1.  **アプリを開く**
    *   `index.html` をブラウザで開いてください。

2.  **参加者の登録**
    *   画面左上のテキストボックスに参加者の名前を入力します。
    *   1行につき1人の名前を入力してください（最大9名まで）。
    *   入力後、「Update Roulette」ボタンを押すとルーレットに反映されます。

3.  **抽選スタート（公平な抽選）**
    *   画面左下の緑色の **「SPIN!」ボタン** をクリックしてください。
    *   ルーレットが回転し、自動的に止まります。

## 詳細設定（確率とカウント）

「⚙️ Probability Settings」ボタンを押すと、詳細設定画面が開きます。

*   **Weight (確率の重み)**
    *   特定の人の当選確率を上げたい場合、この数値を増やしてください。
    *   数値が大きいほど、ルーレット上のその人のエリアが広くなります。
    *   デフォルトは全員「1」です。

*   **Win Counts (当選回数)**
    *   画面左側のリストに、各参加者が何回当たったかが表示されます。
    *   「RESET」ボタンを押すと、すべてのカウントを0に戻せます。
