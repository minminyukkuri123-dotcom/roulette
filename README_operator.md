# 【取扱注意】オペレーター用 裏マニュアル

このファイルには、一般参加者には秘密にするべき「裏機能」について記載されています。
**絶対に画面共有などで見せないようにしてください。**

## 裏機能：当選者の指定（Rigged Spin）

キーボードの数字キーを使うことで、**確率設定を無視して、特定の順番の人を強制的に当選させる**ことができます。

### 操作方法

1.  参加者リストの順番（上から1番目、2番目…）を確認しておきます。
2.  ルーレットを回すタイミングで、キーボードの **数字キー `1` 〜 `9`** を押します。
    *   `1` を押す → **1番目**の人が必ず当たる
    *   `2` を押す → **2番目**の人が必ず当たる
    *   ...
    *   `9` を押す → **9番目**の人が必ず当たる

### 操作方法2（スマホ等、画面タッチ用：シークレットタップ機能）

スマホなどキーボードがない環境、あるいはより自然に操作したい場合向けの方法です。画面上部の **「ROULETTE」というタイトル文字** を使います。

1.  **操作**: 当てたい人の順番の数だけ、**「ROULETTE」の文字を連続してポン・ポンとタップします**。
    *   タイトルを **1回** タップ → **1番目**の人が必ず当たる
    *   タイトルを **2回** タップ → **2番目**の人が必ず当たる
    *   ...
    *   タイトルを **9回** タップ → **9番目**の人が必ず当たる
2.  **発動**: 最後にタップしてから**約1秒後**に、自動的にルーレットが回り始めます。

> 💡 **コツ**: 参加者数を超えて押しすぎた場合でも順番はループします（例: 3人参加で4回押すと1番目の人が当たります）。途中で操作を間違えた場合は、画面をリロードするか、ルーレットが止まるのを待ってから再度操作してください。

### 注意点

*   **画面上の変化はありません**: 画面には何も表示されず、通常の「SPIN!」ボタンを押したときと同じように回転アニメーションが始まります。
*   **自然な挙動**: 停止位置はターゲットの枠内でランダムにズレるため、不自然に真ん中で止まることはありません。
*   **カウント**: 裏機能で当てた場合も、画面上の「WIN COUNTS」は通常通り加算されます。

### 使い分け

*   **ガチ抽選**: 画面上の緑色の「SPIN!」ボタンをクリック（設定された確率通りに抽選）。
*   **演出・調整**: キーボードの数字キーを使用（意図した人を当選させる）。
