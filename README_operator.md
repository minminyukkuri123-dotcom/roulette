# 【取扱注意】オペレーター用 裏マニュアル

このファイルには、一般参加者には秘密にするべき「裏機能」について記載されています。
**絶対に画面共有などで見せないようにしてください。**

## 裏機能：当選者の指定（Rigged Spin）

キーボードの数字キーを使うことで、**確率設定を無視して、特定の順番の人を強制的に当選させる**ことができます。

### 操作方法

1.  参加者リストの順番（上から1番目、2番目…）を確認しておきます。
2.  ルーレットを回すタイミングで、キーボードの **数字キー `1` 〜 `9`** を押します。
    *   `1` を押す → **1番目**の人が必ず当たる
    *   `2` を押す → **2番目**の人が必ず当たる
    *   ...
    *   `9` を押す → **9番目**の人が必ず当たる

### 操作方法2（スマホ・タブレット用：音量ボタン）

スマホなどキーボードがない環境向けに、音量ボタンを使った方法も用意されています。

1.  **準備**: ブラウザの制限を解除するため、ページを開いた後、**必ず画面のどこかを一度タップ**してください。
2.  **操作**: 当てたい人の順番の数だけ、**端末の音量ボタン（上下どちらでも可）を連続してポン・ポンと押します**。
    *   音量ボタンを **1回** 押す → **1番目**の人が必ず当たる
    *   音量ボタンを **2回** 押す → **2番目**の人が必ず当たる
    *   ...
    *   音量ボタンを **9回** 押す → **9番目**の人が必ず当たる
3.  **発動**: 最後にボタンを押してから**約1秒後**に、自動的にルーレットが回り始めます。

> 💡 **コツ**: 押しすぎた場合でも順番はループします（例: 3人参加で4回押すと1番目の人が当たります）。途中で操作を間違えた場合は、画面をリロードするか、ルーレットが止まるのを待ってから再度操作してください。

### 注意点

*   **画面上の変化はありません**: 画面には何も表示されず、通常の「SPIN!」ボタンを押したときと同じように回転アニメーションが始まります。
*   **自然な挙動**: 停止位置はターゲットの枠内でランダムにズレるため、不自然に真ん中で止まることはありません。
*   **カウント**: 裏機能で当てた場合も、画面上の「WIN COUNTS」は通常通り加算されます。

### 使い分け

*   **ガチ抽選**: 画面上の緑色の「SPIN!」ボタンをクリック（設定された確率通りに抽選）。
*   **演出・調整**: キーボードの数字キーを使用（意図した人を当選させる）。
