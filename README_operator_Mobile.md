# 【取扱注意】オペレーター用 裏マニュアル (スマホ版)

このファイルには、一般参加者には秘密にするべき「裏機能」について記載されています。
**絶対に画面共有などで見せないようにしてください。**

## 裏機能：当選者の事前セット（Rigged Spin）

スマホなどキーボードがない環境、あるいはより自然に操作したい場合向けの方法です。画面上部の **「ROULETTE」というタイトル文字** を使います。

### 操作方法（シークレットタップ・セット機能）

1.  **ターゲットのセット**: 当てたい人の順番の数だけ、**「ROULETTE」の文字を連続してポン・ポンとタップします**。
    *   タイトルを **1回** タップ → **1番目**の人をターゲットにセット
    *   タイトルを **2回** タップ → **2番目**の人をターゲットにセット
    *   ...
    *   タイトルを **9回** タップ → **9番目**の人をターゲットにセット
2.  **発動**: タップし終わってから約1秒待つと、内部にターゲットがセットされます（※自動では回りません）。その後、通常の **緑色の「SPIN!」ボタン** を押すと、セットした人に必ず当たるスピーンが開始されます。

> 💡 **コツ**: 参加者数を超えて押しすぎた場合でも順番はループします（例: 3人参加で4回押すと1番目の人が当たります）。途中で操作を間違えた場合は、画面の「Update Roulette」ボタンを押すか、リロードしてからやり直してください。

### 注意点

*   **画面上の変化はありません**: 画面には何も表示されず（デバッグモード時を除く）、通常の「SPIN!」ボタンを押したときと同じように回転アニメーションが始まります。
*   **自然な挙動**: 停止位置はターゲットの枠内でランダムにズレるため、不自然に真ん中で止まることはありません。
*   **カウント**: 裏機能で当てた場合も、画面上の「WIN COUNTS」は通常通り加算されます。

### 使い分け

*   **ガチ抽選**: 何もセットせずに、そのまま画面上の緑色の「SPIN!」ボタンをクリック（設定された確率通りに抽選）。
*   **演出・調整**: タイトルをタップしてターゲットをセットしてから「SPIN!」ボタンを押す（意図した人を当選させる）。
